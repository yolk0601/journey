#### 前提
+  1、有 node npm
    >
    >  1、安装node
    >> 
    >> 通过homebrew安装: brew install node@8(版本号)
    >> 
    >
    > 2. 查看node 是否安装成功
    >> 
    >> node -v
    >>
    >> npm -v 


+  2、安装好 vue vue-cli
    >
    > 1、 npm install vue
    >
    > 2、npm instal vue-cli


#### 启项目
1. vue init webpack(可选) project-name
2. cd project-name
3. npm run dev
